# MediBook Portal

A modern hospital appointment booking and management system built with React and TypeScript.

Live: https://prathamrathod14.github.io/medibook-portal/

## Features

- **Patient Registration & Login** - Secure authentication for patients
- **Doctor Directory** - Browse and search available doctors by specialty
- **Appointment Booking** - Easy scheduling with preferred doctors
- **Appointment Management** - View, reschedule, or cancel appointments
- **Medical Records** - Access to patient health records
- **Responsive Design** - Works on desktop, tablet, and mobile devices

## Tech Stack

- **Frontend:** React 18, TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Routing:** React Router
- **State Management:** React Context / Hooks

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/medibook-portal.git

# Navigate to project directory
cd medibook-portal

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
